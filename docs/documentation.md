## Overview
University Admissions Bot (Sweety) is a Python-based chatbot designed to answer common admission-related queries such as courses, eligibility, fees, hostel, placements, and contact details.

## Objective
To automate university enquiry handling and provide quick, accurate responses to students and parents.

## System Requirements
- Python 3.x
- Command Line Interface

## How It Works
- Takes user input from terminal
- Matches keywords with predefined categories
- Returns relevant university information
- Provides personalized responses

## Modules Used
- datetime (for time-based greeting)

## Limitations
- Keyword-based matching
- No database or AI model
